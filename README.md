# 👻 Nightmare Games

## 📖 Sobre o Projeto

Nightmare Games é um site fictício de venda de jogos de terror desenvolvido utilizando apenas HTML5 e CSS3.

O objetivo do projeto é apresentar uma interface temática inspirada em jogos de horror, contendo um catálogo de jogos, sistema de login e cadastro de usuários.

---

## 🎮 Funcionalidades

- Página inicial com destaque para a loja.
- Catálogo de jogos de terror.
- Página de login.
- Página de cadastro.
- Botão "Saiba Mais" com links externos para informações dos jogos.
- Menu lateral presente em todas as páginas.
- Interface responsiva e temática de terror.
